**Card** — the base surface; a lit-from-within ink panel with a hairline edge.

```jsx
<Card padding={28}>…</Card>
<Card interactive>Hover lifts + glows ember</Card>
<Card glow>Selected / active state</Card>
```

Props: `interactive` (hover lift), `glow` (force ember halo), `padding`, `sheen`, `as`. Compose StatCard, lists, forms inside it.
