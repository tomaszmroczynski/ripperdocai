**CapabilityMeter** — branded progress/gauge; ember track with a glowing leading node ("a signal travelling a connection").

```jsx
<CapabilityMeter label="Knowledge coverage" value={72} />
<CapabilityMeter value={40} tone="info" height={6} showValue={false} />
```

Props: `value`, `max`, `label`, `showValue`, `height`, `tone` (`ember|up|info`).
