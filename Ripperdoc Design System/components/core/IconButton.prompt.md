**IconButton** — square, icon-only action; use in toolbars, card headers, composers. Always pass `label`.

```jsx
<IconButton label="Settings" variant="ghost"><i data-lucide="settings-2" /></IconButton>
<IconButton label="Run" variant="primary" size="lg"><i data-lucide="zap" /></IconButton>
```

Variants/sizes mirror `Button`. Ghost is the default for dense UI.
